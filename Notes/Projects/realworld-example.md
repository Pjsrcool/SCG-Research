**realworld-example**\
fork: https://github.com/Pjsrcool/realworld-example

**Installation**
1. Clone the repo. Stay in main branch
2. Run `yarn install`

**Run without build**
1. run `yarn start`

**To build and run the server**
1. Run `yarn build`
2. Start the server with `serve -s build`

**Notes**
- Fails to convert to es5 under proxy
